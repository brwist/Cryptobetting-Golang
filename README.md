# Technical spect

Go,Postgresql,ORM, Environment ,Docker

## Install dependencies

```sh
go list -m -u all
```

## Run application

```sh
go run main.go
```

## Build application

```sh
go build -v ./...
```

## Test application

```sh
go test -v ./...
```

## Containerize Docker

```sh
docker-compose build
docker-compose up
```
